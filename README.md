# JSExpertMax Gesture Controller - Semana JS Expert 7.0

Projeto desenvolvido na Semana JS Expert organizada pelo Erick Wendel

## Pre-reqs
- Este projeto foi criado usando Node.js v19.6
- Diferente do que foi instruído na proposta inicial do projeto, que indicava fazer em ambiente Unix (Linux), aqui ele foi feito todo em ambiente Windows.
